# Calculadora de IMC com JavaFX

Aplicacao desktop para calcular o IMC, classificar o resultado e praticar os primeiros conceitos de JavaFX em um projeto util.

## Repositorio

[calculadora-imc](https://github.com/p-rcorreia/calculadora-imc.git)

## Objetivo

Construir uma interface grafica simples para receber peso e altura, calcular o Indice de Massa Corporea e exibir a classificacao correspondente.

## Conceitos praticados

- Orientacao a objetos
- Eventos em JavaFX
- Controles de formulario
- Validacao de entrada
- Separacao entre regra de negocio e interface

## Funcionalidades planejadas

- Campo para peso
- Campo para altura
- Botao para calcular
- Exibicao do valor do IMC
- Classificacao do resultado
- Tratamento de entradas invalidas
